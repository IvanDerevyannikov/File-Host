Error 405 - Method Not Allowed

The request method is not supported for this resource

Possible reasons:

  • Using POST method where only GET is allowed
  • Attempting to use PUT or DELETE on read-only resource
  • Incorrect HTTP method for the API endpoint
  • Missing or incorrect CORS configuration
  • Web application firewall blocking the method

What you can do:

  • Check if you're using the correct HTTP method (GET, POST, etc.)
  • Review the API documentation for supported methods
  • Try using a different HTTP method
  • Contact the website administrator if this persists
